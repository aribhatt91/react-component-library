# OTP Input
A generic production grade OTP input

## Functional Requirements

- Customisable Length - Length of OTP eg 4 digit, 6 digit etc.
- Allows only numbers
- Auto-focus on next digit cell on typing
- Error state
- Maskable fields (password-like)

# Non-functional Requirement

- Performance - Ability to handle
- Accessibility - Should be accessible, tabbable
- Security
- Scalability
